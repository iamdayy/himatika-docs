# 🔐 Ganti Email & Password

Anda dapat mengubah alamat email dan password akun HIMATIKA Anda kapan saja untuk menjaga keamanan akun.

---

## 1. Ganti Email

Fitur ini berguna jika Anda ingin menggunakan email baru untuk menerima notifikasi, OTP, atau Magic Link.

### Langkah-langkah:

1. Buka halaman **Profil** Anda.
2. Klik tombol **Ganti Email** atau akses langsung melalui URL `.../change-email`.
3. Masukkan **alamat email baru** yang ingin Anda gunakan.
4. Klik **Kirim**.
5. Sistem akan mengirimkan **kode OTP** ke email baru Anda.
6. Masukkan kode OTP pada halaman verifikasi.
7. Jika kode OTP valid, email Anda akan diperbarui.

::u-alert{variant="soft" color="info"}
#title
💡 Penting
#description
- NIM Anda **tidak bisa diubah** karena merupakan identitas unik di sistem.
- Pastikan email baru yang Anda masukkan **aktif dan bisa diakses**, karena semua notifikasi akan dikirim ke email tersebut.
- Jika Anda menggunakan login **Google OAuth**, email yang terhubung juga bisa berubah.
::

---

## 2. Ganti Password

Fitur ini untuk mengubah password login akun Anda. Anda harus mengetahui password lama untuk melakukan perubahan.

### Langkah-langkah:

1. Buka halaman **Profil** Anda.
2. Klik tombol **Ganti Password** atau akses langsung melalui URL `.../change-password`.
3. Isi formulir berikut:

| Field | Keterangan |
|:---|:---|
| **Password Lama** | Masukkan password Anda yang saat ini aktif |
| **Password Baru** | Masukkan password baru (minimal 8 karakter) |
| **Konfirmasi Password Baru** | Ulangi password baru untuk memastikan tidak typo |

4. Klik **Ganti Password**.
5. Jika berhasil, Anda akan **otomatis di-logout** dan harus login kembali dengan password baru.

::u-alert{variant="solid" color="warning"}
#title
⚠️ Lupa Password Lama?
#description
Jika Anda lupa password lama, gunakan fitur **[Lupa Password](/panduan-anggota/login-reset-password)** di halaman login untuk mereset password melalui email. Atau minta admin untuk melakukan [Reset Password Manual](/panduan-admin/kelola-anggota).
::

---

## 3. Tips Keamanan

- 🔑 Gunakan password yang **unik dan kuat** (kombinasi huruf besar, kecil, angka, dan simbol).
- 🔄 **Ganti password secara berkala** (setiap 3-6 bulan).
- 🚫 **Jangan bagikan password** Anda kepada siapapun, termasuk pengurus.
- 📱 Pertimbangkan menggunakan **login Google** untuk keamanan tambahan.
