# ✏️ Mengelola Profil

Profil Anda adalah identitas Anda di dalam sistem. Menjaga data profil agar tetap akurat sangat penting untuk kelancaran komunikasi dan administrasi keanggotaan.

### Melihat Profil Anda

Untuk melihat detail data diri yang tersimpan di sistem:
1.  Login ke akun Anda.
2.  Di pojok kanan atas, klik pada **nama Anda** atau **foto profil** Anda.
3.  Pilih menu **"Profil Saya"** atau **"Lihat Profil"**.
4.  Anda akan diarahkan ke halaman yang berisi semua data Anda, mulai dari NIM, nama, angkatan, hingga kontak.

### Memperbarui Profil

Jika ada data yang perlu diubah (misalnya nomor telepon atau alamat), ikuti langkah berikut:

1.  Masuk ke halaman **Profil Saya** seperti cara di atas.
2.  Cari dan klik tombol **"Edit Profil"** atau ikon pensil (✏️).
3.  Formulir akan menjadi aktif dan Anda bisa mengubah data pada kolom yang tersedia.

    ![Contoh halaman edit profil](https://via.placeholder.com/800x600.png/E2E8F0/94A3B8?text=Gambar:+Halaman+Edit+Profil)
    *Ganti dengan screenshot halaman edit profil Anda.*

4.  Setelah selesai mengubah data, pastikan Anda menekan tombol **"Simpan Perubahan"**. Jika tidak, data yang Anda ubah tidak akan tersimpan.

::alert{type="warning"}
**Data yang Tidak Bisa Diubah:** Beberapa data penting seperti **Nama Lengkap** dan **NIM** mungkin dikunci dan tidak bisa Anda ubah sendiri untuk menjaga validitas data. Jika terdapat kesalahan pada data ini, silakan hubungi **Divisi Sekretaris** atau **Admin Sistem** untuk meminta bantuan perubahan.
::
