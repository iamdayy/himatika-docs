# 📱 Presensi Mandiri (Scan QR)

Fitur **Presensi Mandiri** memungkinkan anggota dan tamu untuk melakukan check-in kehadiran secara mandiri menggunakan kamera smartphone mereka. Panitia akan menampilkan QR Code acara, dan Anda cukup scan untuk mencatat kehadiran.

---

## 1. Cara Mengakses

Ada beberapa cara untuk mengakses halaman Presensi Mandiri:

- **Dari Dasbor**: Klik tombol besar **🔲 Scan QR** yang ada di pojok kiri bawah halaman dasbor.
- **Dari URL langsung**: Buka `.../agendas/scan`.

::u-alert{variant="soft" color="info"}
#title
💡 Perlu Login
#description
Anda harus **sudah login** terlebih dahulu sebelum bisa menggunakan fitur presensi mandiri. Sistem perlu mengetahui identitas Anda untuk mencatat kehadiran.
::

---

## 2. Cara Melakukan Presensi

1. Buka halaman **Presensi Mandiri**.
2. Browser akan meminta **izin akses kamera** — klik **"Izinkan"**.
3. Arahkan kamera ke **QR Code** yang ditampilkan oleh panitia di lokasi acara.
4. Sistem akan **otomatis mendeteksi** QR Code.

### Hasil Scan

| Hasil | Tampilan | Keterangan |
|:---|:---|:---|
| ✅ **Berhasil** | Layar hijau + ikon centang + suara sukses | Kehadiran Anda tercatat! Poin keaktifan otomatis masuk |
| ❌ **Gagal** | Alert merah + suara error | QR Code tidak valid, Anda tidak terdaftar, atau sudah pernah presensi |

### Setelah Berhasil Scan
- Layar akan menampilkan pesan **"Hadir!"** beserta informasi agenda.
- Anda akan **otomatis diarahkan** ke halaman detail agenda setelah 2 detik.
- **Poin keaktifan** otomatis ditambahkan ke akun Anda.

---

## 3. Troubleshooting

| Masalah | Solusi |
|:---|:---|
| Kamera tidak muncul | Pastikan Anda mengizinkan akses kamera di browser. Cek **Pengaturan → Izin → Kamera** |
| QR Code tidak terdeteksi | Pastikan pencahayaan cukup, dan posisikan QR Code di tengah viewfinder |
| "Gagal memproses QR Code" | Pastikan Anda sudah terdaftar sebagai peserta/panitia di agenda tersebut |
| "Sudah presensi" | Anda sudah melakukan scan sebelumnya. Tidak perlu scan ulang |

---

## 4. Perbedaan dengan Scan Admin

| | Presensi Mandiri (Anggota) | Scan Admin (Panitia) |
|:---|:---|:---|
| **Siapa yang scan** | Peserta/Panitia scan QR acara | Panitia scan tiket peserta |
| **Arah scan** | Anggota → QR Code acara | Panitia → QR Code tiket |
| **Akses** | `/agendas/scan` | `/administrator/agendas/[id]/scan` |
| **Hak akses** | Semua anggota login | Hanya panitia/admin |
