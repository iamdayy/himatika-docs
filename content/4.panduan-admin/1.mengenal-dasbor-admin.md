# 🛡️ Dasbor Administrator

Area ini khusus untuk pengurus harian dan admin sistem. Di sini Anda dapat mengelola seluruh aspek operasional aplikasi.

## Akses Admin

1. Login dengan akun yang memiliki peran **Administrator** atau **Pengurus**.
2. Anda akan melihat menu sidebar yang lebih lengkap dibandingkan anggota biasa.

## Struktur Sidebar Admin

- **Dashboard (Statistik)**: Ringkasan jumlah anggota, agenda aktif, dan pengunjung.
- **Anggota & Pengurus**: Manajemen data induk mahasiswa dan struktur organisasi.
- **Agenda**: Manajemen acara, pendaftaran, dan tiket.
- **Pencapaian**: Input data prestasi anggota.
- **Berita & Galeri**: Pengelolaan konten publikasi.
- **Tanda Tangan**: Verifikasi surat dan dokumen.
- **Pesan**: Kotak masuk dari formulir "Hubungi Kami".
- **Config**: Pengaturan global aplikasi (Visi Misi, Carousels).
- **Audit Logs**: Catatan aktivitas sistem.

## Statistik Utama

Di halaman utama dasbor admin, Anda dapat memantau:

- **Total Anggota Terdaftar**
- **Pengunjung Website** (Harian/Bulanan)
- **Agenda Aktif**
- **Audit Logs**: Merekam aktivitas penting (login, edit data) untuk transparansi.
