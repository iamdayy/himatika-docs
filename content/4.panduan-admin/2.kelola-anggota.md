# 👥 Kelola Anggota (Members)

Menu **Anggota** adalah tempat di mana Anda mengelola basis data seluruh mahasiswa Teknik Informatika yang terdaftar di sistem.

## 1. Verifikasi Anggota Baru

Setiap mahasiswa yang baru mendaftar tidak akan langsung aktif. Mereka harus diverifikasi terlebih dahulu untuk memastikan bahwa mereka benar-benar mahasiswa aktif.

### Langkah-langkah Verifikasi

1. Buka menu **Anggota** > **Verifikasi**.
2. Anda akan melihat daftar pendaftar baru dengan status `Pending`.
3. Periksa data mereka:
    - **NIM**: Pastikan sesuai dengan format NIM prodi.
    - **Nama**: Sesuaikan dengan data akademik.
4. Klik tombol **Terima (`Approve`)** untuk mengaktifkan akun mereka.
    - *Akun yang diterima akan bisa login dan mengakses layanan.*
5. Klik **Tolak (`Reject`)** jika data tidak valid.

## 2. Manajemen Data Anggota

Anda dapat melihat dan mengedit data anggota yang sudah terdaftar.

1. Buka menu **Anggota** > **Daftar Anggota**.
2. Gunakan fitur **Pencarian** untuk mencari berdasarkan Nama atau NIM.
3. Klik pada **Nama Anggota** untuk melihat detail profil mereka.
4. **Edit Data**: Admin dapat membantu anggota mengubah data sensitif (seperti NIM atau Email) jika terjadi kesalahan input.

## 3. Reset Password Manual

Jika anggota mengalami masalah "Lupa Password" dan tidak bisa mengakses email pemulihan:

1. Cari akun anggota tersebut di daftar.
2. Klik tombol **Opsi** (titik tiga) atau masuk ke detail profil.
3. Pilih **Reset Password**.
4. Sistem akan membuat password sementara. Berikan password ini kepada anggota tersebut secara aman.

## 4. Ekspor Data

Anda dapat mengunduh data seluruh anggota dalam format Excel/CSV untuk keperluan arsip atau pelaporan.

- Klik tombol **Ekspor Data** di pojok kanan atas tabel anggota.
