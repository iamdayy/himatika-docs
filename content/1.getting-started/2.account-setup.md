# Registrasi & Login

Untuk mengakses fitur-fitur layanan HIMATIKA seperti pendaftaran acara dan manajemen profil, Anda perlu memiliki akun terlebih dahulu.

## Mendaftar Akun Baru

Jika Anda adalah mahasiswa aktif Teknik Informatika ITSNU Pekalongan dan belum memiliki akun, ikuti langkah berikut:

1. Buka halaman **Register** di aplikasi.
2. Masukkan **NIM (Nomor Induk Mahasiswa)** Anda.
    * *Sistem akan mengecek apakah data NIM Anda sudah terdaftar di database anggota.*
3. Jika NIM ditemukan, lengkapi formulir pendaftaran:
    * **Email**: Gunakan email aktif (disarankan email institusi jika ada).
    * **Password**: Buat kata sandi yang kuat (Minimal 8 karakter, kombinasi huruf besar, kecil, dan angka).
    * **Konfirmasi Password**: Masukkan ulang kata sandi.
4. Klik tombol **Daftar**.
5. Selesai! Anda akan diarahkan ke halaman login.

::alert{type="warning"}
**NIM Tidak Ditemukan?**
Jika NIM Anda tidak ditemukan atau muncul pesan error, silakan hubungi pengurus HIMATIKA bagian Kaderisasi atau Admin untuk verifikasi data Anda.
::

## Masuk ke Aplikasi (Login)

Setelah berhasil mendaftar, Anda dapat masuk ke dalam aplikasi:

1. Buka halaman **Login**.
2. Masukkan **NIM** dan **Password** yang telah Anda buat.
3. Klik tombol **Masuk**.

### Lupa Password?

Jika Anda lupa kata sandi, silakan hubungi Admin untuk melakukan reset password. Fitur reset password mandiri via email akan segera hadir.

## Keamanan Akun

* Jangan pernah membagikan password Anda kepada siapapun.
* Pastikan untuk **Logout** (Keluar) jika Anda mengakses aplikasi menggunakan perangkat umum/publik.
