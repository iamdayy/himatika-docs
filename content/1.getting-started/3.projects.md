# 🚀 Proyek

Halaman **Proyek** adalah etalase karya inovatif anggota HIMATIKA. Di sini ditampilkan semua proyek yang telah dan sedang dikerjakan oleh anggota sebagai portofolio digital organisasi.

---

## 1. Melihat Daftar Proyek

Buka halaman **Proyek** melalui menu navigasi utama. Anda akan melihat daftar proyek dalam bentuk **kartu-kartu** yang menampilkan:

| Elemen | Keterangan |
|:---|:---|
| **Gambar** | Thumbnail atau screenshot proyek |
| **Judul** | Nama proyek |
| **Kategori** | Badge kategori proyek (Web, Mobile, IoT, dll) |
| **Status Progress** | Tingkat peyelesaian: *Not Started*, *Starting*, *On Progress*, *Complete* |
| **Tags** | Label teknologi atau topik (misalnya: `Vue.js`, `Python`, `Machine Learning`) |
| **Tanggal** | Tanggal proyek dimulai |
| **Anggota Tim** | Avatar anggota yang terlibat |
| **Status Publish** | Tanggal dipublikasikan (atau *not published*) |

### Filter & Pencarian
Anda bisa memfilter proyek berdasarkan:
- **Pencarian** teks (judul proyek)
- **Tags** — Filter berdasarkan teknologi/topik
- **Kategori** — Filter berdasarkan jenis proyek
- **Urutan** — Sort berdasarkan Date, Title, Published, atau Progress

---

## 2. Mendaftarkan Proyek Baru

Setiap anggota yang telah login dapat mendaftarkan proyek karya mereka.

### Langkah-langkah:

1. **Login** ke akun HIMATIKA Anda.
2. Buka halaman **Proyek**.
3. Anda akan melihat banner ajakan **"Daftarkan Proyek Karyamu"** di bagian atas halaman.
4. Klik tombol **`Daftarkan Proyek`**.
5. Isi formulir pendaftaran proyek:

| Field | Keterangan |
|:---|:---|
| **Judul** | Nama proyek Anda |
| **Deskripsi** | Penjelasan tentang proyek |
| **URL** | Link ke proyek (GitHub, website live, dll) |
| **Gambar** | Upload screenshot atau logo proyek |
| **Kategori** | Pilih kategori proyek |
| **Tags** | Tambahkan label teknologi (misal: `React`, `Node.js`) |
| **Progress** | Persentase penyelesaian proyek (0-100%) |
| **Anggota Tim** | Pilih anggota HIMATIKA yang terlibat |

6. Klik **Submit** untuk mengajukan proyek.

::u-alert{variant="soft" color="info"}
#title
💡 Info Penting
#description
Proyek yang baru didaftarkan belum langsung terlihat di halaman publik. Admin perlu mereview dan **mempublikasikan** proyek Anda terlebih dahulu.
::

---

## 3. Detail Proyek

Klik judul proyek pada kartu untuk melihat halaman detail yang berisi:

- **Deskripsi lengkap** proyek.
- **Bar progress** visual.
- **Daftar anggota tim** lengkap.
- **Link eksternal** ke repository atau live demo.
- **Tanggal dibuat** dan **tanggal dipublikasikan**.

---

## 4. Manfaat Fitur Proyek

- 🎯 **Showcase** — Menjadi portofolio karya anggota HIMATIKA kepada publik.
- 💪 **Kolaborasi** — Mengasah kemampuan kerja tim dan manajemen proyek.
- 📚 **Dokumentasi** — Merekam perjalanan teknis organisasi dari masa ke masa.
- 🏆 **Poin** — Kontribusi proyek memberikan poin keaktifan bagi anggota.

---

## 5. Pengelolaan Proyek (Admin)

Admin memiliki kemampuan tambahan untuk mengelola proyek:

- **Review** — Memeriksa proyek yang baru diajukan.
- **Publish/Unpublish** — Mempublikasikan atau menyembunyikan proyek.
- **Edit** — Memperbarui informasi proyek.
- **Hapus** — Menghapus proyek yang tidak relevan.
